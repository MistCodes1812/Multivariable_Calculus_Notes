Base cases and recursive cases

Every recursive function needs a base case that terminates the recursion and a recursive case that makes progress toward it. Forgetting the base case is the single most common cause of a stack overflow in student code.

Deep Dive

Why recursion and induction are the same idea

If you’ve done proof by induction, recursive functions should feel familiar: the base case matches your base case, and the recursive step matches your inductive step. This isn’t a coincidence — it’s the same logical structure applied to computation instead of proof.
